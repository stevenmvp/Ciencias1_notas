# Proyectos - Ciencias de la Computación

## 📋 Descripción

Esta sección contiene proyectos integradores que aplican múltiples conceptos aprendidos en clase.

## 🚀 Proyectos Propuestos

### Proyecto 1: Sistema de Gestión de Estudiantes
**Nivel:** Principiante-Intermedio
**Conceptos:** Estructuras, archivos, búsqueda
**Descripción:** Crear un programa que gestione información de estudiantes (nombre, edad, calificaciones)

**Funcionalidades:**
- Agregar estudiantes
- Buscar por nombre o ID
- Mostrar calificaciones
- Guardar en archivo
- Cargar desde archivo

**Tecnologías:** C++ con STL

---

### Proyecto 2: Visualizador de Algoritmos
**Nivel:** Intermedio-Avanzado
**Conceptos:** Algoritmos, visualización, interfaz
**Descripción:** Crear una aplicación que visualice cómo funcionan los algoritmos de ordenamiento

**Funcionalidades:**
- Generar arreglo aleatorio
- Visualizar paso a paso
- Mostrar comparaciones y intercambios
- Estadísticas de rendimiento

**Tecnologías:** C++ con gráficos (SDL2/SFML)

---

### Proyecto 3: Compilador Simple
**Nivel:** Avanzado
**Conceptos:** Parsing, análisis léxico, árboles
**Descripción:** Crear un compilador básico para un lenguaje sencillo

**Fases:**
1. Análisis léxico (tokenización)
2. Análisis sintáctico (parsing)
3. Generación de código

**Tecnologías:** C++ estándar

---

## 📈 Recomendaciones

1. Comienza con proyectos pequeños
2. Lee código de otros programadores
3. Refactoriza después de que funcione
4. Escribe pruebas
5. Documenta tu código
