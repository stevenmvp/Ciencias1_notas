### 1. Algoritmos de Ordenamiento

#### 1.1 Algoritmo de Burbuja (Bubble Sort)
- **Archivo:** `01_algoritmo_burbuja.cpp`
- **Nivel:** Principiante
- **Concepto:** Comparación e intercambio de elementos adyacentes

**Características:**
- Implementación simple y clara
- Comentarios detallados
- Múltiples ejemplos de uso
- Optimización incluida (detección de arreglos ya ordenados)
- Análisis de complejidad

**Complejidad:**
- Mejor caso: O(n)
- Caso promedio: O(n²)
- Peor caso: O(n²)
- Espacio: O(1)

**Cómo usar:**
```bash
g++ -o burbuja 01_algoritmo_burbuja.cpp
./burbuja
```
